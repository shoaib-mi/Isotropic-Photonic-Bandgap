The "generating-data" folder contains C++ scripts designed to generate data under various conditions. These simulations solve coupled integro-differential equations for a three-level V-type atom embedded in an isotropic photonic bandgap environment. A time step of 1e−5 was used to ensure accuracy in the numerical integration. For each simulation case, the output consists of three CSV files containing the complex probability amplitudes for each of the atomic levels. 
Once the raw data is generated, the scripts located in the "post-processing" folder are used to calculate the desired quantum metrics. These metrics include: Quantum Fisher Information Matrix (QFIM), Sigma function, Coherence, Hilbert-Schmidt speed, and Level populations.
Due to the large volume of the resulting measurement files, the codes in the "sampling-data" folder are employed to reduce the dataset size through appropriate sampling techniques. Finally, the scripts within the plotting-data folder are used to generate high-quality figures suitable for publication in academic journals.
